# Seppo - huippuhyvä tekstieditori

Tervetuloa **Seppoon**, intuitiivinen, kevyt (ei ominaisuuksia) ja tehokas(?) tekstieditori, joka on suunniteltu nostattamaan koodaus- ja kirjoituskokemuksesi ennennäkemättömälle tasolla. Luotu tunteitta ja taidoitta, Seppo tarjoaa saumattoman kirjoitusympäristön suoraan päätteessäsi.


## Ominaisuudet

- **Cross-platform Compatibility**: Toimii seuraavilla: Windows, macOS, and Linux.
- **File Management Ease**: Open, edit, and save files effortlessly.
- **Undo/Redo**: (Almost) Full undo and redo capabilities to make editing worry-free.
